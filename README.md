# AiDetective: AI-Driven Scam Detection for Meta Ads

## Overview
AiDetective is an AI-driven system designed to detect and classify scam advertisements within Meta's advertising ecosystem. The system integrates data preprocessing, large language models (LLMs), and classification logic to identify fraudulent ads.

This project is developed as part of an Honours Thesis at Macquarie University.

---

## Features
- Scam detection using LLMs
- Scam classification (phishing, investment, impersonation, etc.)
- Text preprocessing pipeline
- Evaluation using accuracy, precision, recall, and F1-score

---

## System Pipeline
1. Input Meta Ads dataset
2. Preprocessing (clean text, remove noise)
3. Stage 1: Scam detection (LLM)
4. Stage 2: Scam classification (LLM)
5. Output results to CSV
6. Evaluate performance metrics
